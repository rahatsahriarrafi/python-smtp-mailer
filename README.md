<div class="section">
<h2>🚀 Project Overview</h2>

<p class="box">
Python SMTP Mail Sender is a beginner‑friendly email automation project built with Python.  
It demonstrates how to send emails programmatically using the <b>SMTP protocol</b> with Gmail servers.
</p>

<p>
The project explains how email systems work in Python including authentication,
message formatting, and secure transmission using <b>TLS and SSL encryption</b>.
</p>

</div>

<div class="section">
<h2>✨ Features</h2>

<ul>
<li>Send emails using <b>SMTP protocol</b></li>
<li>Custom <b>subject and message</b></li>
<li>Secure email sending with <b>TLS / SSL encryption</b></li>
<li>Simple <b>command line interaction</b></li>
<li>Beginner friendly Python implementation</li>
<li>Demonstrates both <b>SMTP (587)</b> and <b>SMTP_SSL (465)</b></li>
</ul>
</div>

<div class="section">
<h2>🛠 Technologies Used</h2>

<ul>
<li>Python 3</li>
<li>smtplib</li>
<li>SMTP Protocol</li>
<li>Gmail SMTP Server</li>
</ul>
</div>

<div class="section">
<h2>⚙ How It Works</h2>

<ol>
<li>User enters sender email and receiver email.</li>
<li>The script connects to Gmail SMTP server.</li>
<li>User authentication is done using Gmail App Password.</li>
<li>Email subject and message body are formatted.</li>
<li>Email is sent securely using TLS or SSL.</li>
</ol>
</div>

<div class="section">
<h2>🔐 Security Notice</h2>

<p class="box">
⚠️ Never upload your real email password to GitHub.  
Always use a <b>Google App Password</b> when working with SMTP programs.
</p>
</div>
